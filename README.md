## 🎯 จุดประสงค์ในการเข้าศึกษาต่อ

ฉันมีความมุ่งมั่นที่จะเข้าศึกษาต่อที่ **วิทยาลัยพณิชยการบางนา สาขาเทคโนโลยีสารสนเทศ** โดยมีเป้าหมายหลักดังนี้:

* **พัฒนาทักษะด้าน IT:** เพื่อพัฒนาและต่อยอดความรู้ด้านเทคโนโลยีสารสนเทศให้ก้าวทันยุคปัจจุบัน
* **มุ่งสู่การเป็นนักพัฒนาระบบ:** ต้องการเข้าใจโครงสร้างพื้นฐานของระบบซอฟต์แวร์ ทั้งในส่วนของหน้าบ้าน (Frontend) และหลังบ้าน (Backend)
* **ประยุกต์ใช้ในสายอาชีพ:** นำความรู้ที่ได้จากวิทยาลัยพณิชยการบางนาไปประกอบอาชีพเป็น Programmer หรือ Developer ที่มีประสิทธิภาพในอนาคต

## 📚 รายวิชาที่กำลังศึกษา: Backend Development

การเรียนรู้ในส่วนของ Backend เพื่อสร้างระบบหลังบ้านที่แข็งแกร่ง รองรับการทำงานของแอปพลิเคชันอย่างมีประสิทธิภาพ

### หัวข้อย่อยที่เน้นการเรียนรู้
1.  **Node.js & Express:** การสร้าง Web Server และ RESTful API
2.  **Database Management:** การจัดการฐานข้อมูล (SQL / NoSQL)
3.  **Authentication:** ระบบสมัครสมาชิก ล็อกอิน และความปลอดภัย (JWT)
